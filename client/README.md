# Клиентский кабинет (код)

Здесь лежит **вся прикладная логика** личного кабинета клиента (`/client/*`): UI-компоненты, server actions и SSR-контекст.

Маршруты Next.js App Router по-прежнему объявлены в `app/client/` и `app/api/client/*` — так устроен фреймворк; страницы только импортируют модули отсюда (`@/client/...`).

| Папка | Назначение |
|-------|------------|
| `components/` | Клиентские React-компоненты кабинета |
| `actions/` | Server Actions (логин, выход, активация инвайта) |
| `server/context.ts` | Загрузка сессии и данных кабинета для SSR и API routes |

Общие вещи (UI kit `components/ui`, `src/server/nest-*`, схемы Zod) остаются в корне репозитория.
